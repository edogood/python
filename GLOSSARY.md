# GLOSSARY.md — Glossario minimo (da estendere solo se serve)

- **Public API surface**: l’insieme di simboli pubblici pensati per l’uso da parte dell’utente (funzioni/classi/metodi non “interni”).
- **Signature**: forma ufficiale di chiamata (parametri, default, *args/**kwargs).
- **Determinismo**: a parità di input (e seed) ottieni lo stesso output.
- **Side effect**: un effetto collaterale (mutare un oggetto, scrivere su disco, fare I/O).
- **Lazy evaluation**: il calcolo avviene solo quando “materializzi” il risultato.
- **Eager evaluation**: il calcolo avviene subito.
- **Surface contract (SDK enorme)**: criterio esplicito e ripetibile che definisce cosa documentare come “API principale” quando l’SDK genera migliaia di classi/DTO.
- **Mini-test**: assert breve che valida l’esempio mostrato.
